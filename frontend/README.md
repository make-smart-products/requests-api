# Frontend — Личный кабинет

React + Vite + TypeScript UI для Requests API.

## Запуск

```bash
# Терминал 1 — backend
cd ..
go run ./cmd/api

# Терминал 2 — frontend
cd frontend
npm install
npm run dev
```

Откройте http://localhost:5173

## Страницы

- `/login` — вход
- `/register` — регистрация клиента
- `/` — обзор
- `/applications` — заявки
- `/profile` — профиль
- `/notifications` — уведомления

## Тестовый вход

- `admin@requests.local` / `admin12345`
